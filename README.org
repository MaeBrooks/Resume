* Hello! This is my latex setup for my resume!

** Requirements:

- `tlmgr`
- `xelatex`

** Building:

#+BEGIN_SRC bash
cc nob.c -o nob
#+END_SRC

**Re-running `nob` will cause it to rebuild on changes**
```sh
./nob
```
